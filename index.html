Simon Says Game Rules (Digital Version)

Objective: Repeat the sequence of colors displayed by the game.

Gameplay:

  • The game starts with a single color in the sequence.
  • The player must click the colors in the exact order shown.
  • Each round, a new color is added to the sequence, making it longer and more challenging.

Mistakes: If the player clicks the wrong color, the game ends, and their score is displayed.

Scoring: The player's score is determined by the number of correct sequences they can follow without making a mistake.
